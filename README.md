# Laravel 8 CRUD Tanpa Reload 
Laravel 8 CRUD Tanpa Reload

Untuk mengikuti, Aplikasi ini telah didokumentasikan sebagai artikel di [Xsis Academy Blog](https://blog.xsis.academy).

Set up

untuk set up project ini, pertama clone repositiory

$ git clone https://github.com/brahmantiodiaz/CRUD_laravel_tanpa_reload.git
Pindah direktori

$ cd CRUD_Laravel_tanpa_reload
Then install dependencies using Composer
Install dependensi menggunakan Composer

$ composer install

Jalan kan migration

$ php artisan migrate

Jalan kan server menggunakan command

$ php artisan serve

Lalu kunjungi http://127.0.0.1:8000/category

Dibuat dengan
Laravel - The PHP Framework For Web Artisans.

Mysql - A relational database management system.
